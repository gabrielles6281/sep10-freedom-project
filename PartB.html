My idea for the Future of Animation

In the future, I want 3D models to become easier to make.

I brainstormed an idea for a program that would greatly improve the process of 3D modeling.

For the program to work, you would make the front and the backside of what you wanted to become a 3d model. There would be a tool for automatic 3D modeling. The program would analyze both sides of what you rew and would automaticall formatting it into a 3D model.

There would be an option to choose the proportions of the 3D model as well. You would also be able to continue drawing on what you made after the 3d model was created in case you wanted to change anything about it. The layers made for the drawing would stay so you dont have to worry about the different layers of the drawing messing up the model either.
